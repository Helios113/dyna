# Dyna code base


> [!NOTE]

> When we are ready for release, we will highlight which branch and features are ready.
> If you want to contribute, please raise an issue or submit a pull request. There is no standard way to do this yet.



## Dyna
Framework and implementation of dynamic path iterative transformer models.
  This servers as a backbone for DynaLLMs and for DynaDiffusion models.
